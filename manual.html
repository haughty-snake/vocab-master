VocabMaster 사용 매뉴얼

효과적인 영어 단어 학습을 위한 완벽 가이드

중요: 데이터 저장 방식 이해하기

⚠️ 반드시 알아야 할 사항

VocabMaster는 localStorage를 사용하여 학습 데이터를 저장합니다.

이는 다음을 의미합니다:

  • 데이터는 현재 브라우저에만 저장됩니다 - 다른 기기, 다른 브라우저에서는 학습 기록이 보이지 않습니다.
  • 브라우저 데이터를 삭제하면 학습 기록도 삭제됩니다 - 캐시/쿠키 삭제 시 주의하세요.
  • 시크릿/프라이빗 모드에서는 저장되지 않습니다 - 일반 브라우저 모드를 사용하세요.
💡 localStorage란?

localStorage는 웹 브라우저에 내장된 저장소로, 웹사이트별로 최대 5~10MB의 데이터를 저장할 수 있습니다. 서버에 데이터를 전송하지 않아 개인정보가 안전하게 보호되며, 인터넷 연결 없이도 데이터가 유지됩니다.

장점: 빠른 속도, 오프라인 지원, 개인정보 보호, 무료

단점: 기기 간 동기화 불가, 브라우저별 독립 저장

다른 기기에서 학습을 이어가려면?

Export/Import 기능을 사용하세요!

  1. 기존 기기에서: 설정(톱니바퀴) → "학습 데이터 내보내기" 클릭 → JSON 파일 저장
  2. 새 기기에서: 설정(톱니바퀴) → "학습 데이터 가져오기" 클릭 → 저장한 파일 선택
  3. 데이터가 자동으로 병합됩니다 (기존 데이터 + 가져온 데이터)

데이터 압축 기능

🗜️ LZ-String 압축

대용량 학습 데이터를 효율적으로 저장하기 위한 압축 기능입니다.

  • 압축률: 데이터 유형에 따라 50~90% 용량 절감
  • 자동 감지: 기존 JSON 데이터와 압축 데이터 모두 자동 인식
  • 무손실: 압축/해제 시 데이터 손실 없음
  • 안전 변환: 변환 중 데이터 손실 방지 메커니즘 내장

사용 방법: 설정 → "데이터 압축 사용" 토글 활성화

🛡️ 안전한 형식 변환

압축 설정 변경 시 다음 안전 메커니즘이 작동합니다:

  • 변환 중 화면 잠금 (작업 완료까지 대기)
  • 원본 데이터 보존 (임시 공간에 먼저 저장)
  • 데이터 검증 후 적용 (손상 시 자동 롤백)
  • 저장 공간 부족 시 기존 형식 유지

저장소 용량 관리

💾 용량 모니터링

설정 화면에서 현재 저장소 사용량을 실시간으로 확인할 수 있습니다.

  • 사용량 표시: 프로그레스 바와 수치로 현재 용량 확인
  • 경고 표시: 70% 이상 시 노란색, 90% 이상 시 빨간색
  • 압축률 표시: 데이터 압축 시 절감 효과 확인
📊 사용량과 압축률 차이

데이터 사용량과 압축률에 표시된 용량이 다른 이유:

  • 데이터 사용량: 전체 저장소 사용량 (메인 + 백업 + 설정 + 기타)
  • 압축률 용량: 메인 데이터만 표시 (백업 제외)

VocabMaster는 데이터 안전을 위해 메인 + 백업 이중 저장 방식을 사용합니다. 따라서 실제 저장소 사용량은 메인 데이터의 약 2배가 됩니다.

⚠️ 용량 제한 정책

저장소가 85%를 초과하면 새 단어 추가가 제한됩니다.

  • 85% 한계: 시스템 안정성과 학습 상태 저장 공간 확보
  • 자동 차단: 용량 초과 시 단어 추가/가져오기 차단
  • 사전 경고: 대량 가져오기 시 예상 용량 미리 계산

용량 부족 해결 방법:

  1. 데이터 압축 활성화 (50~90% 절감)
  2. 불필요한 카테고리/단어 삭제
  3. 데이터 내보내기 후 정리

멀티탭 동기화

🔄 실시간 동기화

같은 브라우저에서 여러 탭을 열어 사용해도 학습 데이터가 자동으로 동기화됩니다.

동기화 원리

  • Storage 이벤트 감지: 다른 탭에서 데이터 변경 시 자동 감지
  • 스마트 병합: 학습 상태 충돌 시 더 높은 상태 유지
  • 실시간 UI 업데이트: 진행률 바, 카테고리 상태 자동 갱신

사용 시나리오

상황 동작
탭 A에서 단어 "암기 완료" 표시 탭 B의 진행률이 자동 업데이트
탭 A: "학습 중", 탭 B: "암기 완료" 둘 다 "암기 완료"로 통일 (높은 상태 유지)
탭 A에서 사용자 카테고리 추가 탭 B에서 새 카테고리 자동 표시
💡 참고 사항
  • 같은 브라우저 내에서만 동기화됩니다 (Chrome ↔ Chrome)
  • 다른 브라우저나 기기 간에는 Export/Import를 사용하세요
  • 시크릿/프라이빗 모드에서는 동기화되지 않습니다

가져오기 (Import) - 병합 방식

✅ 스마트 병합 기능

Import 시 기존 데이터를 덮어쓰지 않고 지능적으로 병합합니다:

현재 상태 가져온 상태 결과
새 단어 학습 중 학습 중 (높은 상태 유지)
학습 중 암기 완료 암기 완료 (높은 상태 유지)
암기 완료 학습 중 암기 완료 (높은 상태 유지)

이렇게 하면 여러 기기에서 학습한 내용이 합쳐지며, 한 번 암기 완료한 단어는 절대 되돌아가지 않습니다!

플래시카드 모드

전통적인 암기 카드 방식입니다. 카드를 클릭하면 뒤집어집니다.

  • TTS 음성 읽기: 단어/예문 버튼 클릭 시 음성 재생
  • TTS 자동/수동: 카드 넘길 때 자동 읽기 설정 가능

키보드 단축키:

  • Space 또는 Enter - 카드 뒤집기
  • ← - 이전 카드
  • → - 다음 카드
  • 1 - 암기 완료 표시
  • 2 - 학습 중 표시

블링크 모드

단어가 빠르게 지나가며, 순간적인 인식과 기억력을 훈련합니다.

  • 속도 조절: 0.5초 ~ 3초
  • 표시 옵션: 영어만 / 뜻만 / 둘 다
  • 반복 횟수: 1회 ~ 5회
  • TTS 자동 읽기: 단어 표시 시 자동 음성 재생

퀴즈 모드

4개의 선택지 중 정답을 고르는 방식으로 학습 효과를 테스트합니다.

  • 문제 유형: 뜻 맞추기 / 단어 맞추기 / 혼합
  • 문제 수: 10 / 20 / 50 / 100문제
  • 다의어 지원: [품사]뜻 형태로 표시

학습 방법의 과학적 근거

🧠 능동적 회상 (Active Recall)

플래시카드와 퀴즈 모드는 "능동적 회상" 원리를 활용합니다. 단순히 정보를 읽는 것보다, 기억에서 정보를 "꺼내는" 과정이 장기 기억 형성에 훨씬 효과적입니다.

📚 연구: Karpicke & Roediger (2008) - "The Critical Importance of Retrieval for Learning", Science

🔄 간격 반복 (Spaced Repetition)

단어를 "새 단어 → 학습 중 → 암기 완료"로 분류하여 관리하면, 아직 완벽히 암기하지 못한 단어에 더 집중할 수 있습니다. 이는 간격 반복 학습의 기본 원리입니다.

📚 연구: Cepeda et al. (2006) - "Distributed Practice in Verbal Recall Tasks", Psychological Bulletin

⚡ 빠른 노출 학습 (Rapid Serial Visual Presentation)

블링크 모드의 빠른 단어 노출은 RSVP 기법을 활용합니다. 짧은 시간 동안 집중적으로 노출하면 무의식적 학습이 촉진되고, 단어 인식 속도가 향상됩니다.

📚 연구: Potter (1984) - "Rapid Serial Visual Presentation (RSVP)", Attention and Performance X

🎯 테스트 효과 (Testing Effect)

퀴즈를 통해 자신을 테스트하면, 단순 복습보다 50% 이상 더 나은 장기 기억을 형성합니다. 틀린 문제도 학습의 일부이며, 오히려 더 강한 기억을 만들어냅니다.

📚 연구: Roediger & Karpicke (2006) - "Test-Enhanced Learning", Psychological Science

📝 맥락적 학습 (Contextual Learning)

모든 단어에 예문과 해석을 함께 제공합니다. 단어를 문맥 속에서 학습하면 의미 이해와 실제 사용 능력이 크게 향상됩니다.

📚 연구: Nation (2001) - "Learning Vocabulary in Another Language", Cambridge University Press

학습 상태 관리

세 가지 상태

상태 표시 의미
새 단어 ⚪ 회색 원 아직 학습하지 않은 단어
학습 중 🟡 노란 원 학습을 시작했지만 아직 암기 중인 단어
암기 완료 🟢 초록 원 완벽히 암기한 단어

상태 변경 방법

  • 단어 목록에서: 상태 원을 클릭하면 순환 (새 단어 → 학습 중 → 암기 완료)
  • 플래시카드에서: "알아요" → 암기 완료, "몰라요" → 학습 중
  • 퀴즈에서: 정답 → 암기 완료, 오답 → 학습 중

사용자 정의 카테고리

📁 나만의 단어장 만들기

기본 제공되는 카테고리 외에 나만의 카테고리를 만들어 원하는 단어를 추가할 수 있습니다. 사용자 정의 카테고리는 점선 테두리와 사용자 뱃지로 구분됩니다.

카테고리 만들기

  1. 홈 화면에서 "카테고리 추가" 카드를 클릭합니다.
  2. 카테고리 이름, 아이콘, 색상을 선택합니다.
  3. "저장" 버튼을 클릭하면 새 카테고리가 생성됩니다.

카테고리명 수정

  1. 사용자 정의 카테고리의 ⚙️ 버튼을 클릭합니다.
  2. 단어 관리 모달 상단의 카테고리명 옆 ✏️ 버튼을 클릭합니다.
  3. 새 이름을 입력하고 "저장" 버튼을 클릭합니다.

단어 추가 및 관리

  1. 사용자 정의 카테고리의 우측 상단 ⚙️ 버튼을 클릭합니다.
  2. "단어 추가" 버튼으로 개별 단어를 추가하거나, "파일 가져오기"로 여러 단어를 한 번에 추가할 수 있습니다.
  3. 추가된 단어는 목록에서 수정하거나 삭제할 수 있습니다.

영영사전 조회 기능

Free Dictionary API를 사용하여 영어 단어 정보를 자동으로 불러올 수 있습니다.

  • 사전 버튼: 단어 입력창 옆의 "사전" 버튼 클릭
  • 자동 입력 항목:
    • 발음기호 (IPA)
    • 품사 (영어 → 한글 자동 변환: noun → 명사, verb → 동사 등)
    • 뜻 (영어 정의)
    • 예문 (영어)
  • 다의어 지원: 여러 뜻이 있는 단어는 자동으로 뜻 입력창이 생성됩니다 (최대 10개)
품사 변환 목록
noun → 명사verb → 동사adjective → 형용사
adverb → 부사preposition → 전치사conjunction → 접속사
interjection → 감탄사pronoun → 대명사determiner → 한정사

발음기호 조회 기능

발음기호만 따로 조회할 수 있습니다.

  • 조회 버튼: 발음 입력창 옆의 "조회" 버튼 클릭
  • 단어 입력창에 영어 단어를 먼저 입력한 후 조회합니다.
  • 영영사전 API에서 발음기호(IPA)를 가져와 자동 입력합니다.

* 두 기능 모두 Free Dictionary API를 사용하며, 인터넷 연결이 필요합니다.

다의어 등록

하나의 단어에 여러 뜻을 등록할 수 있습니다.

  • "+ 뜻 추가" 버튼으로 여러 뜻을 등록
  • 각 뜻마다 품사, 예문, 해석을 개별 입력 가능
  • 학습 시 [품사]뜻 형태로 표시됨

💡 뜻 비교 방식

뜻은 쉼표(,)로 구분하여 비교합니다.

예시: 기존 뜻이 "사과, 바나나"이고 "사과, 포도"를 입력하면 → "사과, 바나나, 포도"가 됩니다.

중복된 뜻은 자동으로 제거됩니다.

대량 가져오기 (Bulk Import)

10,000개 이상의 단어도 빠르게 가져올 수 있습니다.

⚡ 배치 처리 성능
  • 최적화된 처리: 메모리에서 일괄 처리 후 한 번에 저장
  • 빠른 속도: 10,000개 단어도 수 초 내 완료
  • 프로그레스 바: 진행률 실시간 표시 (예: 5,000 / 10,000)
  • 중간 취소: 가져오기 중 닫기 버튼으로 취소 가능
사용 방법
  1. 사용자 정의 카테고리 선택
  2. "파일 가져오기" 버튼 클릭
  3. JSON 또는 CSV 파일 선택
  4. 프로그레스 바로 진행률 확인
  5. 완료 시 결과 토스트 표시 (신규/업데이트/다의어 개수)
✅ 데이터 백업에 포함

사용자 정의 카테고리와 추가한 단어들은 Export/Import 시 자동으로 포함됩니다. 다른 기기에서도 나만의 단어장을 그대로 사용할 수 있습니다!

카테고리 공유

🔗 단어장 공유하기

사용자 정의 카테고리를 다른 사람들과 공유할 수 있습니다. 설정 메뉴의 "사용자 정의 카테고리 공유" 섹션을 이용하세요.

카테고리 내보내기

  1. 설정(⚙️) 버튼을 클릭합니다.
  2. "사용자 정의 카테고리 공유" 섹션에서 "카테고리 내보내기"를 클릭합니다.
  3. 내보낼 카테고리를 선택합니다 (전체 선택 가능).
  4. "내보내기" 버튼을 클릭하면 JSON 파일이 다운로드됩니다.

공유받은 카테고리 가져오기

  1. 설정(⚙️) 버튼을 클릭합니다.
  2. "사용자 정의 카테고리 공유" 섹션에서 "카테고리 가져오기"를 클릭합니다.
  3. 공유받은 JSON 파일을 선택합니다.
  4. 가져온 카테고리가 사용자 정의 카테고리에 추가됩니다.

  3. 틀린 것에 집중: "학습 중" 상태의 단어를 필터링하여 집중 학습하세요.
  4. 예문과 함께: 단어만 외우지 말고, 예문을 소리 내어 읽어보세요.

자주 묻는 질문 (FAQ)

Q: 다른 컴퓨터에서 접속했는데 학습 기록이 없어요!

A: 정상입니다. localStorage는 브라우저별로 독립적으로 저장됩니다. 기존 기기에서 Export한 파일을 새 기기에서 Import하세요.

Q: 브라우저 캐시를 삭제했는데 기록이 사라졌어요!

A: 캐시/사이트 데이터 삭제 시 localStorage도 함께 삭제됩니다. 중요한 학습 기록은 정기적으로 Export해두세요.

Q: Import할 때 기존 데이터가 덮어써지나요?

A: 아니요! 스마트 병합 방식으로, 더 높은 학습 상태가 유지됩니다. 예: 기존에 "암기 완료"인 단어는 Import 파일에서 "학습 중"이어도 "암기 완료"로 유지됩니다.

Q: 휴대폰에서도 사용할 수 있나요?

A: 네! 반응형 디자인으로 모바일에서도 편리하게 사용할 수 있습니다. 다만 휴대폰과 PC의 학습 기록은 각각 저장되므로, 동기화하려면 Export/Import를 사용하세요.

질문이나 피드백이 있으시면 알려주세요!
